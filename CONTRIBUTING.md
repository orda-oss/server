# Contributing

This repo is source-available for transparency and security auditing purposes.
We do not accept external pull requests at this time.

## Bug reports

If you find a bug, open an issue with steps to reproduce, your platform, and
the server version.

## Security disclosures

Found a security issue? **Do not open a public issue.**

Email [hello@joinorda.com](mailto:hello@joinorda.com) with `[security]` in
the subject line.
